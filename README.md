AI Personal Finance Advisor

📌 Overview

This project is an intelligent financial advisory system that analyzes user spending patterns and provides personalized saving recommendations using Machine Learning.

🚀 Features

* Predict user spending patterns
* Data preprocessing and feature engineering
* Expense analysis (temporal and category-based)
* Personalized saving recommendations
* Interactive web app using Streamlit

🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

📊 Process

1. Data Cleaning and Preprocessing
2. Feature Engineering
3. Model Training for prediction
4. Data Analysis and Visualization
5. Recommendation Generation

📈 Results

* Improved understanding of user spending habits
* Generated personalized financial suggestions

▶️ How to Run

1. Clone the repository
2. Install dependencies
3. Run: `streamlit run app.py`

🌐 Deployment

* Built using Streamlit for interactive UI
